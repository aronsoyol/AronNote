## AronNote

- 专门为传统蒙古文而诞生的竖排纯文本编辑器
- 内嵌蒙古文字体，可以再没有创痛蒙古文字体的环境正常显示
- 可以作为.net 控件调用
- 蒙古文菜单
- 显示UNICODE代码
